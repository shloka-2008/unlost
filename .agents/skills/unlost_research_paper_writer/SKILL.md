# Research Writing Guidelines

Always write using:

- IEEE academic tone
- Third-person perspective
- Formal language
- Proper transitions
- Clear explanations
- Technical accuracy

Do not write like marketing content.

Avoid:
- "Amazing"
- "Revolutionary"
- "Game-changing"
- "Best ever"

Instead use:
- improves
- enhances
- facilitates
- provides
- enables
- supports
- reduces
- optimizes
